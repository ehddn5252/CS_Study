# CS 스터디

## 운영 방식

1. 매 주 5개씩 질문을 선택해서 이에 대해 정리하고 답변 준비를 한다.
2. 정리를 다 하면 각 질문에 맞는 카테고리에 질문을 Pull request한다.
3. 주 1회 정리한 질문에 대해서 서로 면접 방식으로 질문을 한다.(꼬리질문도 )
